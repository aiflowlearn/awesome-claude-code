# 实战任务 5.3: 用 subagent review 自己的项目

## 目标
用之前创建的 security-reviewer agent 审查自己构建的 TODO 工具。

## 操作
在 Claude Code 中输入：

```
用 security-reviewer agent 审查 TODO 工具的全部代码
```

如果 security-reviewer agent 未配置，也可以用自然语言：

```
请审查这个项目中所有代码的安全性、代码质量和最佳实践
```

## 完成标准
输出完整的审查报告。包含发现的问题、严重程度和修复建议。这是课程的最后一个任务——你已经完整体验了从安装 Claude Code 到用它构建并审查一个真实项目的全过程。
