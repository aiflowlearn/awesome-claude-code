# Entity: Claude Code

## Type

AI coding agent / command-line development environment

## Definition

Claude Code is Anthropic’s agentic coding tool for reading, editing, testing, and managing software projects from the terminal.

## Why It Matters

Claude Code can operate across a full repository, follow project instructions from CLAUDE.md, use hooks and MCP servers, and preserve workflow knowledge through skills and memory.
