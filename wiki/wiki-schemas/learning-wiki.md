# Learning Wiki Schema

Use this CLAUDE.md schema for a learning-focused llm wiki.

```
wiki/
├── raw/        # imported lessons, transcripts, docs
├── entities/   # tools, people, products, commands
├── concepts/   # ideas, patterns, mental models
└── notes/      # reflections, summaries, practice logs
```

## Entity Page Fields

- Name
- Type
- One-sentence definition
- Why it matters
- Related concepts
- Practice tasks
- Source links

## Concept Page Fields

- Concept
- Explanation for beginners
- Common mistakes
- Examples
- Related entities
- Review questions
