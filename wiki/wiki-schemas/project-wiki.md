# Project Wiki Schema

Use this schema for a software project knowledge base.

```
wiki/
├── raw/
├── entities/      # services, modules, APIs, jobs
├── concepts/      # architectural patterns and constraints
├── decisions/     # ADR-style notes
└── runbooks/      # operational procedures
```

## Required Links

Every page should link to owners, related modules, source files, and verification commands when available.

## Page Fields

- Purpose
- Current behavior
- Inputs and outputs
- Dependencies
- Failure modes
- Verification
- Related pages
