# my-ai-toolkit - Claude Code 工作规则

## 项目类型
全栈项目

## 技术栈
React, TypeScript, Tailwind CSS

## 工作方式
- 先解释计划，再修改文件。
- 修改前说明会影响哪些文件。
- 遇到不确定需求时先澄清。
- 不自动执行危险命令。

## 验证命令
npm run build && npm run lint

## 个人偏好
先解释计划，再修改文件。改完后必须说明验证方式。不要自动执行危险命令。

## 常用任务
- explain：解释代码结构和关键路径。
- fix：定位问题，给出最小修改。
- review：检查风险、回归和遗漏测试。
- test：运行验证并总结结果。
